# 🍽️ Restaurant Billing System---

## 🚀 Features

* 🧾 Generate bill with GST & packaging
* 🍛 25-item Indian menu
* 🛒 Cart system with quantity updates
* ❌ Remove items from cart
* 🗄️ Store bills in MySQL database
* 📄 PDF receipt generation
* 📊 View bill history

---

## 🛠️ Technologies Used

* Java Swing (GUI)
* MySQL (Database)
* JDBC (Connectivity)
* iText (PDF Generation)

---

## ▶️ How to Run

1. Clone the repository
2. Open in IntelliJ IDEA
3. Add MySQL connector JAR
4. Start MySQL server
5. Run `MainApp.java`

---

## 📸 Output

* Bill generated in UI
* Data stored in MySQL
* PDF receipt created
